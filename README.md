# Inventory Management System

## Overview

Inventory Management System is a Java console application designed to help users manage products inside a small inventory. The system allows users to add products, display product information, view all products, sell products, and restock inventory quantities.

The project was developed using Object-Oriented Programming (OOP) concepts and demonstrates the use of classes, constructors, methods, ArrayLists, loops, and conditional statements.

---

## Features

* Add new products to the inventory
* View detailed information about a specific product
* Display all available products
* Sell products and update quantities
* Restock products when inventory is low
* Input validation through menu options
* Console-based user interface

## Classes Used

### Product Class

The `product` class stores the following information:

* Product name
* Product price
* Product quantity

It also contains methods for:

* Getting product name
* Getting product quantity
* Updating product quantity
* Displaying product information

### Main Class

The `inm` class contains the main menu and handles all inventory operations such as adding, selling, restocking, and viewing products.

---

## Menu Options

1. Add Product
2. Product Info
3. View Products
4. Sell Product
5. Restock Product
6. Exit

---

## How to Run

1. Open the project in your Java IDE.
2. Compile the `inm.java` file.
3. Run the program.
4. Select options from the menu to manage products.
